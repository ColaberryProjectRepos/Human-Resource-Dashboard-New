Step 3 – Measures Table

Measures Table upon creating all the DAX measures. 




Steps :

Create a new Table named – Measures Table.

1. Begin by selecting "Transform data" to access Power Query.

2. Navigate to "Enter Data" within Power Query.

3. Give the table a new name, such as "Measures Table," and confirm by clicking OK.

4. After entering the data, close the Power Query window by selecting "Close & Apply."

DAX Measures added are listed below –

Measures Created are:

·         Total Emp = COUNTROWS('HR Analytics Data')

·         Male = CALCULATE([Total Emp], 'HR Analytics Data'[Gender]= "Male" )

·         % Male = DIVIDE([Male], [Total Emp], 0)

·         Female = CALCULATE([Total Emp], 'HR Analytics Data'[Gender]= "Female" )

·         % Female = DIVIDE([Female], [Total Emp], 0)

·         Due for Promotion = CALCULATE([Total Emp], 'HR Analytics Data'[Promotion Status] = "Due" )

·         Not Due for Promotion = CALCULATE([Total Emp], 'HR Analytics Data'[Promotion Status] = "Not due" )

·         % Due for Promotion = DIVIDE([Due for Promotion], [Total Emp], 0)

·         % Not Due for Promotion = DIVIDE([Not Due for Promotion], [Total Emp], 0)

·         High Performance Rating = CALCULATE([Total Emp], 'HR Analytics Data'[Performance Status]= "High" )

·         Low Performance Rating = CALCULATE([Total Emp], 'HR Analytics Data'[Performance Status]= "Low" )

·         % High Performance Rating = DIVIDE([High Performance Rating], [Total Emp], 0)

·         % Low Performance Rating = DIVIDE([Low Performance Rating], [Total Emp], 0

·         Active Employees = CALCULATE([Total Emp], 'HR Analytics Data'[Employee Attrition] = "Active Employee" )

·         Ex Employees  = CALCULATE([Total Emp], 'HR Analytics Data'[Employee Attrition] = "Ex-Employee" )

·         % Active Employees = DIVIDE([Active Employees], [Total Emp], 0)

·         % Ex Employees  = DIVIDE([Ex Employees], [Total Emp], 0)

·         AverageAge = AVERAGE('HR Analytics Data'[Age])

·         Job Satisfaction % = DIVIDE ( AVERAGE('HR Analytics Data'[JobSatisfaction]),4,0)

·         Attrition Arrow = SWITCH(TRUE(), [% Ex Employees] < 0, UNICHAR(9660), UNICHAR(9650))

·         Attrition Rate Color = IF([% Ex Employees] > 0.1, "#D6272C","#1AAB40")

·         Average Hourly Rate = AVERAGE('HR Analytics Data'[HourlyRate])

·         Average Service Yrs = AVERAGE('HR Analytics Data'[YearsAtCompany])

·         Job Satisfaction Arrow = SWITCH (TRUE(), ([Job Satisfaction %]*100)>50, UNICHAR(9650), ([Job Satisfaction %]*100)<50, UNICHAR(9660), UNICHAR(9654))

·         Job Satisfaction Arrow Color = SWITCH(TRUE(), [Job Satisfaction %] > 0.5, "#1AAB40", [Job Satisfaction %] < 0.5, "#D6272C", [Job Satisfaction %] = 0.5, "#DB7228")
